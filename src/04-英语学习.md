# 常用英文

### 图形学：  
```
normal-法线  
AO-环境光遮蔽  
texture-纹理（材质）  
sample-采样  
density-密度  
PostProcessVolume-后期处理体积  
StaticMesh-静态网格体
```

handy：方便的、便携的、实用的
written：手写的、书写的
companion：伴侣、伙伴、配套手册 / 本子

handy writtencompanion：便携笔记本
