# CMkae语法

## 一、CMake语法概述
- CMake项目的根目录下必须有一个CMakeLists.txt文件，包含了构建项目所需的指令和设置。
- CMake语法类似于函数调用，使用命令和参数来定义构建规则。
- CMake语法支持变量、条件语句、循环等编程结构，使得构建规则可以更加灵活和复杂。
- CMake提供了丰富的内置命令和模块，帮助开发者轻松管理项目的构建过程。

## 二、变量
变量是 CMake 的核心，用于存储路径、文件名或编译器选项。  
- CMake中的变量分为两种
    - CMake提供
    - 用户定义

- CMake的变量名区分大小写

- CMake中的变量在存储时都是字符串

- 变量的定义/赋值：使用 set()。
    ```bash
    # set(变量名 值)
    set(SRC_FILES main.cpp tool.cpp)
    ```

- 变量的引用：直接在代码中使用变量名即可。
    ```bash
    # ${变量名}
    message("Sources are: ${SRC_FILES}")
    ```
